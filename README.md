# Multiple Color Squares

This project is a web application that allows users to view and filter colored squares based on their color. The application features a main color square, thumbnail squares, buttons for filtering colors, and a dropdown menu for selecting colors.

## Features

- Display colored squares in different shades (red, green, and blue)
- Filter and display squares based on selected color
- Change the main color square by clicking on a thumbnail square
- Select colors using buttons or a dropdown menu
- Responsive design with Bootstrap integration

## Technologies Used

- HTML
- CSS
- JavaScript
- Bootstrap (for styling)

## Getting Started

To run the project locally, follow these steps:

1. Clone the repository:

   ```
   git clone https://github.com/DingxDon/ColorSelector.git
   ```

2. Navigate to the project directory:

   ```
   cd multiple-color-squares
   ```

3. Open the `index.html` file in your preferred web browser.

Alternatively, you can view the live demo by visiting [insert your live demo URL here].

## Usage

- By default, all colored squares are displayed in the thumbnails container, and the main color square is set to the first color (#FF0000).
- Click on the "All", "Red", "Green", or "Blue" buttons to filter and display the corresponding colored squares.
- Select a color from the dropdown menu to achieve the same filtering functionality.
- Click on any thumbnail square to update the main color square with the selected color.
